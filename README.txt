Functions:

poly_div : output the quotient polynomial and remainder polynomial resulting from the division of the first input polynomial by the second input polynomial
		a_node : Pointer to a polynomial
		b_node : Pointer to another polynomial

print_poly : output a polynomial to stdout as comma separated pairs of the form (i, ai), ai≠0 ordered on decreasing values of i
		num : Pointer to the polynomial

get_number_input : inputs a polynomial from stdin as comma separated pairs of the form (i, ai), ai≠0 ordered on decreasing values of i
	Returns a new Pointer to the polynomial
